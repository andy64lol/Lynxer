# Lynxer Standard Library

This directory contains documentation for the Lynxer standard library modules.

Modules

- [colorlib](colorlib.md) — ANSI color helpers for terminal output
- [fileIO](fileIO.md) — File reading/writing and utilities
- [http](http.md) — Simple HTTP client
- [js](js.md) — Run JavaScript via Node.js
- [json](json.md) — JSON helpers
- [math](math.md) — Math utilities and constants
- [os](os.md) — OS and filesystem helpers
- [random](random.md) — Random number utilities
- [server](server.md) — Lightweight test HTTP server
- [shell](shell.md) — Run shell commands
- [time](time.md) — Date/time helpers
- [tkinter](tkinter.md) — Simple GUI helpers
- [turtle](turtle.md) — Turtle graphics wrappers
- [typing](typing.md) — String/list/type utilities

Usage

Each module documents available global functions and a short description of behavior and return values.
